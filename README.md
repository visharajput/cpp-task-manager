# cpp-task-manager

A simple console-based task manager written in C++.

## Features
- Add tasks with categories
- View pending & completed tasks
- Mark tasks as completed
- Save/load tasks using `tasks.txt`
- Summary stats for pending/completed tasks

## Concepts Used
- Classes
- Vectors
- File I/O
- Conditionals & loops
